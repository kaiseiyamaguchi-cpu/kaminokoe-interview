# SEOコンテンツ量産戦略

## 概要

- **目的**: 検索流入獲得、長期的な資産構築
- **自動化レベル**: 90%（記事生成はAI、公開は半自動）
- **リスク**: ゼロ（コンテンツは資産になる）
- **期待効果**: 高（6ヶ月後から本格流入）

---

## プラットフォーム選定

| プラットフォーム | メリット | デメリット | 判定 |
|---------------|---------|-----------|------|
| **note** | ドメインパワー強、SEO有利、無料 | 独自ドメインではない | ◎ 初期推奨 |
| **Zenn** | エンジニア向けに強い | 就活層には弱い | △ |
| **独自ブログ** | 完全コントロール | ドメイン育成に時間 | ○ 中期以降 |
| **サービス内** | CVに近い | 初期はドメインパワー弱い | ○ 併用 |

**結論**: 初期はnote中心 → 中期でサービス内ブログ併用

## キーワード戦略

### Tier 1: 企業名 × 面接（ロングテール・大量生産）

**パターン**: `{企業名} 面接 対策`

```
例:
- 「トヨタ 面接 対策」
- 「三菱商事 面接 質問」
- 「ソニー 面接 落ちた」
- 「マッキンゼー ケース面接」
```

**対象企業リスト（初期100社）**:
- 総合商社: 三菱商事、三井物産、伊藤忠、住友商事、丸紅
- コンサル: マッキンゼー、BCG、ベイン、デロイト、アクセンチュア
- 金融: 三菱UFJ、みずほ、野村證券、ゴールドマン、モルガン
- メーカー: トヨタ、ソニー、パナソニック、日立、キーエンス
- IT: Google、Amazon、楽天、サイバーエージェント、リクルート
- その他人気企業: JR東海、電通、博報堂、任天堂、資生堂

**記事構成テンプレート**:
```markdown
# {企業名}の面接対策完全ガイド【2025年最新】

## {企業名}の面接の特徴
- 面接回数・形式
- 重視されるポイント
- 過去の傾向

## よく聞かれる質問10選
1. 志望動機系
2. ガクチカ系
3. 自己PR系
4. 企業理解系
5. 逆質問

## 質問別の回答例と解説
（各質問に対する模範回答）

## 面接官が見ているポイント
## 落ちる人の共通点
## AI面接練習のすすめ
→ 神の声への誘導

## まとめ
```

**生成数**: 100社 × 1記事 = 100記事

## 記事生成パイプライン

### 自動化フロー

```
[キーワードリスト]
    ↓
[記事構成テンプレート選択]
    ↓
[Claude API: 記事本文生成]
    ↓
[品質チェック（自動）]
    ↓
[スプレッドシート: 記事DB]
    ↓
[note/ブログに投稿]
```

### 生成プロンプト（例）

```
あなたは就活のプロです。以下の条件で記事を書いてください。

【企業名】: {企業名}
【記事タイプ】: 面接対策ガイド
【文字数】: 3000-4000字
【トンマナ】: 就活生に寄り添う、具体的、実践的
【必須要素】:
- よく聞かれる質問10選
- 各質問の回答例
- 面接官の視点
- 落ちる人の特徴
【CTA】: AI面接練習ツール「神の声」への誘導（自然に）

※架空の情報は書かない。一般的な傾向として書く。
```

### 品質チェック項目
- [ ] 文字数3000字以上
- [ ] 見出し構造が正しい
- [ ] CTAが含まれている
- [ ] 企業名が正しい
- [ ] 明らかな誤情報がない
