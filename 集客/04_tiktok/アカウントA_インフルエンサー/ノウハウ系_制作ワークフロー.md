**作成日**: 2026年3月7日

---

## 概要

投稿以外は完全自動化する。

```
[1. 台本確定] → [2. 画像選定] → [3. Canva文字挿入] → [4. 固定誘導] → [投稿（手動）]
     ↓              ↓              ↓              ↓
   Claude API    素材フォルダ    テンプレート      CTA固定
```

---

## 1. 台本確定

### 入力
- テーマ（例: 「就活スケジュール」）

### 出力
- 5-7枚分のテキスト

### 自動化方法

**Claude APIプロンプト**:

```
あなたは就活で内定を複数獲得した先輩です。
TikTok用のスライドコンテンツを作成してください。

【テーマ】: {テーマ}

【出力形式】:
1枚目（フック）:
「〇〇って子は絶対コレ!!」形式で、2秒で止まるフック

2-6枚目（本体）:
・1枚につき1ポイント
・テキストは3行以内
・絵文字を適度に使用
・具体的なアクション or 体験談

7枚目（CTA）:
・プロフィール誘導
・「詳しくはプロフから」等

【トンマナ】:
- 友達に話す感じ
- 「〜って子は」「〜な人は」で自分事化
- 盛り気味OK（1を10に盛る）
```

### 台本テンプレート例

**テーマ: 就活スケジュール**

```
[1枚目]
正直やりたいことない😅って子は絶対コレ!!
もう迷わない🔥
```

## 2. 画像選定

### 必要な素材カテゴリ

| カテゴリ | 用途 | 枚数目安 |
|---------|------|---------|
| **駅・電車** | ホーム、足元、車窓 | 20枚 |
| **カフェ** | ドリンク、手元、店内 | 20枚 |
| **オフィス街** | ビル、通り | 10枚 |
| **文房具・ノート** | 勉強感 | 10枚 |
| **スーツ（体のみ）** | 就活感 | 10枚 |
| **黒背景** | テキストメイン用 | 1枚（使い回し） |

### 素材の調達先

| ソース | URL | 特徴 |
|--------|-----|------|
| **Unsplash** | unsplash.com | 高品質、無料 |
| **Pexels** | pexels.com | 無料、日本的な素材もあり |
| **Canva内蔵** | canva.com | テンプレ内で完結 |

### フォルダ構成

```
04_tiktok_instagram/
└── 素材/
    ├── 駅_電車/
    │   ├── 001.jpg
    │   ├── 002.jpg
    │   └── ...
    ├── カフェ/
    ├── オフィス街/
    ├── 文房具/
    ├── スーツ/
    └── 黒背景/
```

### 選定ルール

- 1投稿に同じカテゴリの画像を2枚以上使わない
- 顔が映っている画像は使わない
- 縦長（9:16）にトリミングできる構図を選ぶ

## 3. Canvaでの文字挿入

### テンプレート設計

**サイズ**: 1080 x 1920 px（TikTok縦型）

**レイヤー構成**:
```
[背景画像]
    ↓
[半透明オーバーレイ（黒50%）] ← 文字の視認性UP
    ↓
[テキストボックス（白背景）]
    ↓
[テキスト（黒文字）]
```

### テキストスタイル

| 要素 | フォント | サイズ | 色 |
|------|---------|--------|-----|
| フック（1枚目） | ゴシック太字 | 60-80pt | 黒 |
| 本文 | ゴシック | 40-50pt | 黒 |
| 強調部分 | - | - | 赤 or オレンジ |
| 背景ボックス | - | - | 白（角丸） |

### テンプレート作成手順

1. Canvaで「TikTok動画」サイズを選択
2. 背景に素材画像を配置
3. 半透明の黒オーバーレイを追加
4. 白背景の角丸ボックスを追加
5. テキストを配置
6. **テンプレートとして保存**

### 量産時の流れ

```
1. テンプレートを複製
2. 背景画像を差し替え
3. テキストを台本から貼り付け
4. 微調整（位置、サイズ）
5. ダウンロード（PNG）
```

**所要時間**: 1枚2-3分 × 7枚 = 15-20分/投稿
