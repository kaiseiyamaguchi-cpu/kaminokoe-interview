## 訴求トーン：ギリギリ攻める

**基本方針**
- 「本番で使える」とは言わない
- でも見たら100%わかる
- 最後に「練習でね」を1秒入れて保険

**テクニック**

| 手法 | 例 |
|-----|-----|
| 言い切らない | 「これ見ながら答えたら...わかるよね？」 |
| 視聴者に委ねる | 「どう使うかは、お前ら次第」 |
| 棒読み注釈 | 「※練習用です」（明らかにネタ感） |
| 第三者視点 | 「こういうやつ絶対おるw」 |
| 仮定形 | 「もしこれ画面の横にあったらさ」 |
| 見せるだけ | 言葉で説明しない、画面だけ映す |

## 制作ワークフロー

### 必要素材

1. **AI画面のモックアップ / 実際の画面録画**
   - 神の声の実際の動作画面
   - 面接官の質問が入力され、回答が生成される様子

2. **音声**
   - BGM（TikTokトレンド音源）
   - 面接官の質問音声（合成音声でOK）

3. **テキスト**
   - 冒頭フック
   - 保険ワード
   - CTA

### 制作手順

```
1. 画面録画を撮影（神の声のデモ操作）
   ↓
2. CapCutで編集
   - 冒頭3秒のフック挿入
   - 字幕追加
   - BGM追加
   - 30秒以内にカット
   ↓
3. 保険ワード + CTA追加
   ↓
4. TikTokにアップロード
```

### ツール

| 用途 | ツール |
|-----|-------|
| 画面録画 | OBS / QuickTime |
| 動画編集 | CapCut（無料） |
| 音声合成 | ElevenLabs / VOICEVOX |
| サムネイル | Canva |
