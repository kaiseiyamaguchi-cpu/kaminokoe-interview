## CapCut編集メモ

### レイヤー構成
```
レイヤー1: デモ映像（Meet + 神の声の画面録画）
レイヤー2: 冒頭フックテキスト（0:00-0:03）
レイヤー3: 質問字幕（0:05-0:08）
レイヤー4: ハイライトテキスト（0:10-0:12）
レイヤー5: 保険ワード（0:25-0:28）
レイヤー6: CTAテキスト（0:28-0:30）
レイヤー7: BGM（全編）
```

### フォント設定
```
冒頭・ハイライト・CTA:
  - フォント: ゴシック太
  - 色: 白
  - 縁: 黒（太め）
  - 影: あり

保険ワード:
  - フォント: ゴシック細
  - 色: 白（透明度80%）
  - 縁: なし or 薄め

字幕:
  - フォント: ゴシック
  - 色: 白
  - 縁: 黒
  - 位置: 下部
```
