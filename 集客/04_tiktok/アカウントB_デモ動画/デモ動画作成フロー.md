# デモ動画作成フロー

撮影は1パターンのみ。サムネと文章を変えて10本以上作る。

---

## デモ動画の具体的イメージ

### 画面構成（縦型 9:16）

```
┌─────────────────────────┐
│  神の声               │  ← ヘッダー（見切れてOK）
├─────────────────────────┤
│                         │
│  👔 面接官              │  ← 黄色い吹き出し
│  「自己PRをお願いします」│
│                         │
│                         │
│  💬 こう答えよう        │  ← 青い吹き出し（ここがメイン）
│  「私の強みは粘り強さ    │
│   です。学生時代、      │
│   サークルで100人規模   │
│   のイベントを企画...」  │
│                         │
├─────────────────────────┤
│      [終了]  [クリア]    │  ← フッター
└─────────────────────────┘
```

### 撮影時の見せ方

**ポイント: 「質問が入る → 回答がバッと出る」瞬間を撮る**

```
[0秒]  面接官の質問が表示されている状態
         ↓
[3秒]  AIが回答を生成中（画面に何か動きがある）
         ↓
[5秒]  青い吹き出しで回答がバッと出る ← ここが一番のハイライト
         ↓
[10秒] 回答を3-5秒見せて終了
```

### 見せたい「神」な瞬間

1. **質問の認識**: 「👔 面接官」の黄色い吹き出しに質問が出る
2. **回答の生成**: 「💬 こう答えよう」の青い吹き出しに回答が出る
3. **回答の内容**: 「私の強みは〜」など、そのまま読めば完璧な回答

### 撮影シナリオ（具体的な流れ）

```
撮影開始 ─────────────────────────────────

[0-5秒]
  画面: 「面接の会話を聞いています...」の待機状態
  ※ 実際は事前に質問を録音しておいて再生する

[5-8秒]
  画面: 👔面接官「自己PRをお願いします」が表示される

[8-12秒]
  画面: AIが処理中（少し間がある）

[12-25秒]
  画面: 💬「こう答えよう」の青い吹き出しに回答が表示される
  ※ 長めの回答がバッと出る瞬間を見せる

[25-30秒]
  画面: 回答をゆっくりスクロールして全体を見せる

─────────────────────────────── 撮影終了
```

## 撮影（1パターンだけ）

### 撮影する内容
```
神の声の画面を録画（30秒）

1. 面接官の質問が入る
2. AIがリアルタイムで回答を生成
3. 完璧な回答が表示される
```

これだけ。この30秒の素材を全動画で使い回す。

---

## 撮影の準備

### 事前に用意するもの

1. **質問の音声ファイル**（または自分で話す）
   - 「自己PRをお願いします」
   - 面接官っぽい落ち着いた声で

2. **神の声のプロフィール設定**
   - 強み: 粘り強さ
   - ガクチカ: サークルで100人イベント企画
   - 志望理由: 何でもOK

3. **撮影環境**
   - PC or スマホ
   - 画面録画ソフト（OBS / QuickTime）
   - 通知オフ

---

## 撮影手順（10分で完了）

```
1. 神の声を開く
2. プロフィールを事前入力
3. 「開始」をタップして待機画面へ
4. OBS or QuickTimeで画面録画開始
5. 質問音声を再生（または自分で話す）
6. AIが質問を認識 → 回答を生成
7. 回答が表示されたら3秒待つ
8. 録画停止
```

**注意:**
- 無音で録画（BGMは後から）
- マウスの動きは滑らかに
- 通知オフ

## 動画の構成（共通）

```
[0-3秒]   サムネ / 冒頭フック（テキスト）
[3-28秒]  デモ映像（共通の素材）
[28-30秒] 保険ワード + CTA
```

**変えるのは:**
- 冒頭3秒のテキスト
- 途中の字幕テキスト
- 最後の保険ワード
- BGM

**変えないのは:**
- デモ映像本体（同じ素材を使い回す）

## 編集フロー（1本5分）

CapCutでの作業:

```
1. デモ映像を配置（3秒目から開始）
   ↓
2. 冒頭にテキスト追加（0-3秒）
   - フォント: 太め、白、黒縁
   - 位置: 画面中央
   ↓
3. 途中の字幕追加（必要な場合のみ）
   ↓
4. 最後に保険ワード + CTA追加（28-30秒）
   ↓
5. BGM追加（TikTokトレンド音源）
   ↓
6. 書き出し
```
