# 撮影台本（案3: 1人2役）

---

## セットアップ

```
デバイスA（面接官役）: スマホ or PC
  - Google Meetに参加
  - 面接官の質問を話す or 録音を再生

デバイスB（就活生役）: PC
  - Google Meetに参加（カメラON、マイクミュート）
  - 神の声を起動
  - 画面録画ON
```


### 変化球質問

```

```

## 撮影の流れ（実際の手順）

### 準備（5分）

```
1. Google Meetを開始（1人ミーティング）
2. デバイスAでMeetに参加（面接官役）
3. デバイスBでMeetに参加（就活生役）
4. デバイスBで神の声を起動
5. ウィンドウを左右に配置
6. 画面録画スタート
```

### 撮影（5分）

```
[0秒]
  デバイスA: 面接官役として質問を話す
  「では、自己PRをお願いします。」

[5秒]
  デバイスB: 神の声が質問を認識
  → 画面に「👔 面接官: 自己PRをお願いします」が表示

[8秒]
  デバイスB: 神の声がAI処理中

[12秒]
  デバイスB: 回答がバッと表示される
  → 「💬 こう答えよう: 私の強みは...」

[25秒]
  デバイスB: 回答を少しスクロールして見せる

[30秒]
  録画停止
```

## 神の声の事前設定

撮影前にプロフィールを入力しておく:

```
名前: 山田
志望企業: 株式会社〇〇
強み: 粘り強さ、課題解決力
ガクチカ: サークルで100人規模のイベントを企画運営。
         集客に苦戦したが、SNS戦略を見直して目標達成。
志望理由: 御社のミッションに共感。
         特に〇〇事業で挑戦したい。
```

これで「自己PR」と聞けば、ちゃんとした回答が出る。

## 面接官の話し方のコツ

リアルに聞こえるように:

```
× 「自己PRお願いします」（早口）
○ 「では、自己PRを、お願いします。」（ゆっくり、間を取る）

× 棒読み
○ 少し落ち着いた、低めのトーン

× 明るすぎる
○ ビジネスっぽい、淡々とした感じ
```
