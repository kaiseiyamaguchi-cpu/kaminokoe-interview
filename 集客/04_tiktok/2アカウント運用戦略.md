## 最重要原則

### 冒頭2秒で手を止めさせる

**これが全て。** 2秒で止まらなければ、どんなに良いコンテンツでも見られない。

#### フックの型

| 型 | 例 | 効果 |
|---|-----|------|
| **共感煽り** | 「正直やりたいことない😅って子は絶対コレ!!」 | 自分のことだ→止まる |
| **断定** | 「もう迷わない🔥」 | 解決策がある→止まる |
| **数字** | 「内定3社取った方法」「質問TOP10」 | 具体性→止まる |
| **秘密暴露** | 「面接官が本当に見てること」 | 知りたい→止まる |
| **衝撃** | 「面接中にAIがカンペ出してくれる」 | え？→止まる |
| **ネガティブ** | 「これやると落ちます」 | 怖い→止まる |

#### 1枚目のデザイン原則

```
✓ テキストは大きく、太く
✓ 背景はシンプル（黒 or 写真ぼかし）
✓ 絵文字で目を引く（🔥😅💪）
✓ 2行以内に収める
✓ 「〜って人は」「〜な子は」で自分事化
```

#### NG例

```
✗ 「就活について解説します」→ 抽象的すぎ
✗ 「面接対策」→ フックがない
✗ 長文 → 読む前にスクロール
✗ 地味なデザイン → 目に止まらない
```
