# TikTok/Instagram 自動運用戦略

## 概要

- **目的**: バイラル獲得、若年層へのリーチ、LP誘導
- **自動化レベル**: 80%（台本AI生成 + 動画半自動生成）
- **リスク**: 低〜中（垢BANリスクあり、複数アカウントで分散）
- **期待効果**: 高（バイラル時は爆発的リーチ）

---

## プラットフォーム特性

| 項目 | TikTok | Instagram Reels |
|-----|--------|-----------------|
| **ユーザー層** | 10代後半〜20代前半 | 20代〜30代 |
| **就活生親和性** | ◎ 高い | ○ やや高い |
| **アルゴリズム** | フォロワー数関係なくバイラル可能 | フォロワー数がやや影響 |
| **動画長さ** | 15秒〜3分（60秒推奨） | 15秒〜90秒 |
| **投稿頻度** | 1日1〜3本 | 1日1本 |

**結論**: TikTokメイン、Instagramはクロスポスト

## コンテンツ戦略

### カテゴリ1: 面接あるある/共感系（バイラル狙い）

**目的**: 拡散、認知
**自動化度**: 90%

```
例:
- 「面接で『最後に質問ありますか？』って聞かれた時の脳内」
- 「志望動機考えてる時の自分 vs 面接中の自分」
- 「ガクチカがバイトしかない時のごまかし方」
- 「面接官『緊張してる？』 → 内心パニック」
```

**フォーマット**:
- テキストオーバーレイ + 背景動画（フリー素材）
- ミーム形式
- 15〜30秒

### カテゴリ2: 面接Tips/教育系

**目的**: 価値提供、信頼構築
**自動化度**: 85%

```
例:
- 「面接官が『この子落とそう』と思う瞬間TOP5」
- 「志望動機で絶対言ってはいけないNGワード」
- 「逆質問で『特にありません』以外の答え方」
- 「自己PRを30秒で完璧に伝える方法」
```

**フォーマット**:
- AI音声 + テキストスライド
- または顔出しなしのスクリーン録画風
- 30〜60秒

### カテゴリ3: AIツール紹介/デモ

**目的**: サービス認知、LP誘導
**自動化度**: 70%

```
例:
- 「AIに面接練習させたら的確すぎた」
- 「このAI、面接官より厳しい件」
- 「面接対策AI使ったら内定取れた話」
- 「面接中にAIがリアルタイムでサポートしてくれるアプリ」
```

**フォーマット**:
- 実際のアプリ画面録画
- Before/After形式
- 30〜60秒

### カテゴリ4: 質問回答/リクエスト対応

**目的**: エンゲージメント、コミュニティ
**自動化度**: 50%

```
例:
- 「コメントで多かった『コンサル志望の志望動機』を解説」
- 「この質問への回答、AIに評価させてみた」
```

**フォーマット**:
- コメント返信動画
- デュエット/ステッチ

## 自動化パイプライン

### 台本生成（自動）

```
[ネタDB] → [Claude API] → [台本生成] → [レビュー] → [動画制作キュー]
```

**プロンプト例**:
```
就活生向けTikTok動画の台本を作成してください。

【テーマ】: 面接で落ちる人の特徴
【フォーマット】: リスト形式（TOP5）
【長さ】: 45秒で読める分量
【トンマナ】: 友達に話す感じ、カジュアル、共感を誘う
【フック】: 最初の3秒で興味を引く

【出力形式】:
- フック（3秒）
- 本題（35秒）
- CTA（7秒）
```

### 動画生成（半自動）

**オプション1: AI音声 + スライド**
```
[台本] → [ElevenLabs/VOICEVOX] → [音声生成]
                                      ↓
[Canva/CapCut] → [テンプレート適用] → [動画出力]
```

**オプション2: AIアバター**
```
[台本] → [HeyGen/D-ID] → [アバター動画生成] → [編集] → [出力]
```

**オプション3: 画面録画テンプレート**
```
[台本テキスト] → [自動テキストアニメーション] → [BGM追加] → [出力]
```

**推奨**: 初期はオプション1（コスト低）、反応見てオプション2検討

### 投稿（半自動）

```
[動画ファイル] → [Later/Planoly] → [スケジュール投稿]
```

## リスク管理

### 垢BANリスク
- **対策1**: 複数アカウント運用（メイン + サブ2つ）
- **対策2**: 自動化ツールの過度な使用を避ける（投稿は手動orスケジューラー）
- **対策3**: コミュニティガイドライン遵守

### 炎上リスク
- **対策1**: 特定企業・個人への批判を避ける
- **対策2**: 投稿前のNGワードチェック
- **対策3**: コメント監視（週1回）

### コンテンツ枯渇リスク
- **対策1**: ネタDBを常時100以上ストック
- **対策2**: トレンドキャッチアップ（週1回）
- **対策3**: コメント/リクエストからのネタ収集
