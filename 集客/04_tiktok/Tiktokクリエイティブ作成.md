# TikTokクリエイティブ作成フロー

## 前提条件

- **対象コンテンツ**: 複数画像のノウハウ公開・経験談共有系に限定
- **構成フォーマット**: `台本10本.md` の形式（スライド1〜7構成）
- **制作ツール**: OpenClaw + Canva

---

## 必要な素材

### 1. タイトル画像（サムネイル）
| 項目 | 内容 |
|------|------|
| 用途 | スライド1（フック） |
| 作成方法 | バズっている投稿のサムネをAIで加工・変換 |
| 必要数 | 数十枚をストック |
| ステータス | **未作成** |

### 2. 本文用画像（スライド2〜7）
| 項目 | 内容 |
|------|------|
| 用途 | 各スライドの背景 |
| 作成方法 | 汎用背景を使い回し |
| ステータス | **ストック済** |

### 3. テキストブロックテンプレート
| 項目 | 内容 |
|------|------|
| 用途 | 各スライドにテキストを配置するテンプレ |
| 作成方法 | Canvaで作成 |
| ステータス | **未作成** |

## OpenClawへの指示内容

### インプット
1. 台本（台本10本.md形式）
2. タイトル画像（ストックから選択）
3. 本文用画像（ストックから選択）
4. Canvaテンプレート

### タスク
```
各スライド（1〜7）に対して:
  - 対応する画像を選択
  - 台本のテキストを画像上に配置
  - Canvaテンプレートのフォーマットに従う
```

### アウトプット
- 完成したスライド画像 7枚/本 × 投稿数
