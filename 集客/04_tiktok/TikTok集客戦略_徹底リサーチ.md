## 1. TikTokアルゴリズム攻略（2025-2026年最新仕様）

### 1.1 アルゴリズムの基本構造

TikTokの推薦エンジンは**2段階システム**を採用:

1. **Seed Group配信**: 投稿直後、200-500人の「シードグループ」にテスト配信
2. **段階的拡大**: 反応が良ければより大きなオーディエンスへ順次拡大

> フォロワー数に関係なく、初回投稿でも数十万回再生を獲得できる民主的な仕組み

### 1.2 最重要指標ランキング

| 優先度 | 指標 | 説明 |
|--------|------|------|
| 1 | **視聴完了率** | 最後まで見られたか（90%以上が理想） |
| 2 | **シェア数** | 「バイラルエンジン」として最強のシグナル |
| 3 | **保存数** | 再視聴意図を示す高価値シグナル |
| 4 | **コメント数** | エンゲージメントの質を示す |
| 5 | **いいね数** | 2026年は相対的に重要度低下 |
| 6 | **複数回再生** | ループ視聴は人気動画として認識される |

### 1.3 For Youページに載る条件

- **視聴完了率が高い**（8割以上の完走率が目標）
- **初速が良い**（投稿後1-2時間のエンゲージメントが重要）
- **コンテンツの関連性**（フォロワー数より内容重視）
- **制作品質**（照明、音声、編集の意図が明確）
- **SEO最適化**（キャプション、ハッシュタグ、音声認識対応）

### 1.4 最初の3秒のフック設計

**TikTokでバズるかは最初の3秒で99%決まる**

#### 効果的なフックパターン

| パターン | 具体例 |
|----------|--------|
| 断定型 | 「これを知らないと損します」 |
| 数字型 | 「99%の人が間違っている面接の真実」 |
| 疑問型 | 「なぜ内定者は緊張しないのか？」 |
| 衝撃型 | 「面接官が一瞬で不採用を決める瞬間」 |
| 禁止型 | 「絶対に言ってはいけない志望動機3選」 |
| 秘密型 | 「元人事が明かす面接の裏側」 |

#### 冒頭3秒で必要な情報

1. **何の動画なのか**を明示
2. **見るベネフィット**を伝える
3. **答えを最初に提示**（視聴維持率向上）

### 1.5 視聴完了率を高めるテクニック

#### ラブレター構造
1. **What**（何について話すか）を冒頭で明示
2. **Why**（なぜ重要か）で関心を深める
3. **How**（具体的方法）で実用価値を提供

#### 編集テクニック
- **ジェットカット**: 無駄な間をカットしてテンポアップ
- **2秒ルール**: 2秒以上同じ画面を続けない
- **画面切り替え**: 頻繁に視覚的変化を入れる
- **テロップ必須**: 音なし視聴者対応 + 視覚的刺激

## 3. 自動化・効率化パイプライン

### 3.1 AI台本生成 → 動画制作フロー

```
1. 悩み出し → ChatGPT
2. 投稿アイディア → ラッコキーワード
3. 台本文字起こし → Vrew
4. 画像生成 → Canva / AI生成ツール
5. キャプション作成 → ChatGPT
6. ハッシュタグ選定 → display purposes
7. 編集・字幕 → CapCut
8. 投稿・分析 → TikTokアナリティクス
```

### 3.2 顔出しなしで伸びるパターン

| パターン | ツール | 特徴 |
|----------|--------|------|
| **AIアバター** | Elai.io | 140種類以上、60言語対応 |
| **VTuber風** | Latte（日本発） | ポップなテイスト、TikTok向け |
| **画面録画+テロップ** | CapCut | ツール操作デモに最適 |
| **スライド形式** | Canva | 情報系コンテンツ向け |
| **ナレーション+BGM** | Vrew | 知識系・まとめ系 |

### 3.3 ツール活用

#### CapCut（ByteDance公式）
- 無料の動画編集アプリ
- スマホ・PC・Web対応
- **自動字幕生成**機能が強力
- TikTokとの親和性が高い

#### その他推奨ツール
- **Canva**: サムネイル、スライド作成
- **Vrew**: AI文字起こし、自動テロップ
- **ChatGPT**: 台本、キャプション生成

### 3.4 動画テンプレート活用

TikTokアプリ内「テンプレート」機能:
- 同期、エフェクト配置、アニメーションが事前設定
- 1時間の作業を数分に短縮可能
- CapCut、Canva、Adobe Expressでも利用可
