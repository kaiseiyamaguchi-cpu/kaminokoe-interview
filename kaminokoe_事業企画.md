# Kaminokoe 事業企画書

## 概要

**プロジェクト名**: Kaminokoe（神の声）
**コンセプト**: オンライン面接中にリアルタイムで回答をサジェストするAI（Interview Copilot）
**リポジトリ**: https://github.com/kaiseiyamaguchi-cpu/kaminokoe

---

## 1. プロダクト概要

### 1.1 何をするサービスか

```
オンライン面接（Zoom、Google Meet、Teams等）を受けている最中に、
AIがリアルタイムで「こう答えるといい」をサジェストしてくれるサービス
```

### 1.2 技術スタック

| レイヤー | 技術 |
|----------|------|
| フロントエンド | Next.js（PWA対応） |
| リアルタイム音声 | OpenAI Realtime API（WebRTC） |
| 音声認識 | gpt-4o-transcribe |
| レスポンス生成 | GPT-4o Realtime |
| ホスティング | Vercel |

### 1.3 レスポンス速度

| 指標 | 時間 |
|------|------|
| 最初のトークン | 180〜300ms |
| テキスト表示 | 300ms〜800ms |
| 体感 | **会話のテンポを崩さないレベル** |

### 1.4 使い方

```
1. スマホでKaminokoeを開く
2. 「開始」ボタンを押す
3. スマホをPCの横に置く
4. 面接を受ける
5. 相手の質問に対して、AIが回答案を画面に表示
```

### 5.1 成功事例の分析

| サービス | 最初のフック | マネタイズ |

| Handshake | 大学キャリアセンター連携 | 年1億ドル（AIデータ事業） |

### 5.2 Kaminokoeの展開パス

```
Phase 1: 求職者獲得
├── 面接Copilot無料提供
├── 口コミで広がる（「内定取れた！」）
└── 大学キャリアセンター連携

Phase 2: 初期マネタイズ
├── 送客報酬（アフィリエイト）
├── サブスク課金
└── 企業別パック販売

Phase 3: プラットフォーム化
├── 人材紹介開始（許可取得）
├── 企業向けSaaS（候補者の面接力スコア提供）
└── 人材紹介会社向けツール提供

Phase 4: データビジネス
├── 面接傾向レポート販売
├── AI学習データ提供
└── 労働市場インサイト
```

### 5.3 「面接サポート × 人材紹介」の統合モデル

```
通常の人材紹介:
  紹介 → 面接 → 内定率30% → 成功報酬

Kaminokoeモデル:
  紹介 → 面接（AIサポート付き）→ 内定率60%? → 成功報酬

= 同じ候補者数で2倍の売上
= 候補者も「ここ経由で応募すると受かりやすい」と認識
= 企業も「ここからの候補者は通過率高い」と認識
```

**「内定率を上げられる人材紹介会社」という唯一無二のポジション**

## 8. ASP登録・アフィリエイト開始手順

### 8.1 主要ASP

| ASP | 審査難易度 | 転職案件 | URL |
|-----|-----------|----------|-----|
| A8.net | ◎ 超簡単 | 多い | https://www.a8.net/ |
| afb | ○ 普通 | **高単価多い** | https://www.afi-b.com/ |
| バリューコマース | △ やや厳しい | 多い | https://www.valuecommerce.ne.jp/ |
| アクセストレード | ○ 普通 | 多い | https://www.accesstrade.ne.jp/ |

### 8.2 登録手順

```
1. A8.netに登録（5分）
   - メールアドレス
   - 銀行口座
   - サイトURL（なくてもOK）

2. 案件を探す
   - 管理画面 → プログラム検索
   - 「転職」「就活」で検索
   - 提携申請

3. リンク取得
   - 提携承認後 → 広告リンク作成
   - URLをコピー
   - アプリに設置
```

### 8.3 注意点

| 注意点 | 詳細 |
|--------|------|
| 確定率 | 30-50%を想定（面談完了が条件の場合） |
| 否認リスク | 重複登録、既存会員、条件未達 |
| サイト審査 | 高単価案件は審査あり |
