## Kaminokoe向け推奨

### 第一優先: OfferBox / キミスカ
- **理由**: 登録のみで成果確定、新卒就活生の認知度高
- **単価**: 低い（1,000〜2,000円）
- **対策**: 数で稼ぐ。面接対策完了後に「次はスカウトを待とう」として訴求

### 第二優先: キャリアチケット / キャリアスタート
- **理由**: 高単価（5,000〜10,000円）、面接対策→エージェント予約の導線
- **課題**: 面談完了が成果条件
- **対策**: 「AI面接練習したら、本番のエージェント面談へ」の流れを設計

### 保留: MeetsCompany
- **理由**: イベント参加型、オンラインイベントもあり中間的
- **検討条件**: 就活初期層へのリーチがあれば有効

| 転職（サイト登録） | 1,000〜6,000円 | 会員登録 |

## 送客サービス（アフィリエイトの代替案）

### 概要

アフィリエイトではなく、**B2B契約で直接リードを送客する**ビジネスモデル。
ASP規約の制限を受けず、**一括登録形式が可能**。

### アフィリエイトとの違い

| 項目 | アフィリエイト | 送客サービス |
|------|---------------|--------------|
| 契約形態 | ASP経由 | 企業と直接契約 |
| 一括登録 | ❌ 規約上NG（代理申込扱い） | ✅ 可能 |
| 単価 | 1,000〜15,000円 | **3〜10万円/名** |
| 規約制限 | ASP規約に準拠 | 個別契約次第 |

### 単価（アフィリエイトより大幅に高い）

| 送客先 | 単価 |
|--------|------|
| 説明会・選考動員 | **3〜5万円/名** |
| 理系・体育会系学生 | **5〜10万円/名** |
| 新卒紹介（内定成立時） | **60〜120万円/名** |

※ アフィリエイト（1,000〜15,000円）の **10〜50倍** の単価

### 仕組み

```
ユーザー: Kaminokoeに登録
    ↓
☑ 提携サービスへの情報提供に同意（デフォルトでON）
    ↓
Kaminokoe → 就活サービス各社にリード送客
    ↓
各社から成果報酬（3〜10万円/名）
```

### Kaminokoeでの実装イメージ

```
┌─────────────────────────────────────┐
│ Kaminokoeに登録して面接対策を始める   │
│                                     │
│ ☑ 提携就活サービスからスカウトを     │
│   受け取る（おすすめ）               │
│                                     │
│ [登録して始める]                     │
└─────────────────────────────────────┘
```

→ 同意したユーザーの情報を提携各社に送客

### 主要な送客サービス提携先候補

| 会社 | サービス | 特徴 |
|------|----------|------|
| ネオキャリア | 就職エージェントneo | 年間18万人登録、送客単価3万円〜 |
| レバレジーズ | キャリアチケット | 年間13万人利用 |
| i-plug | OfferBox | 新卒オファー型No.1 |
| グローアップ | キミスカ | スカウト型 |
| unistyle | 送客サービス | G-MARCH以上70%、年間8万人登録 |

### メリット

1. **単価が圧倒的に高い**: アフィリエイトの10〜50倍
2. **一括登録が可能**: ユーザーの同意でまとめて送客
3. **ASP規約の制限なし**: 独自の導線設計が可能
4. **ユーザー体験向上**: 「登録1回で複数サービス利用可能」は価値提供

### デメリット・注意点

1. **直接契約が必要**: 各社との営業・交渉が発生
2. **一定のユーザー数が必要**: 送客サービス提携には規模感が求められる
3. **個人情報取り扱い**: プライバシーポリシー整備が必須

### 規約上の注意（アフィリエイトとの比較）

**アフィリエイトでの一括登録がNGな理由:**
> A8.net規約: 「一括資料請求サイトなど、資料請求を代行するサイトも代理申込に該当するためNG」

**送客サービスがOKな理由:**
- ユーザー本人が「情報提供に同意」している
- 企業間のB2B契約であり、ASP規約の対象外
- 送客サービスとしての契約条件に準拠

## 推奨戦略

### フェーズ1: アフィリエイトでスタート
- 小規模でも始められる
- OfferBox / キミスカの登録型案件から開始
- 単価: 1,000〜2,000円/名

### フェーズ2: 送客サービスへ移行
- ユーザー数が一定規模に達したら提携交渉
- 単価: 3〜10万円/名（10倍以上）
- 一括登録形式の導線を実装

## 次のアクション

### 短期（アフィリエイト）
1. **ASP登録**: A8.net → バリューコマース → アクセストレードの順で登録
2. **案件確認**: 各ASPで下記を検索
   - OfferBox
   - キミスカ
   - キャリアチケット
   - キャリアスタート
   - MeetsCompany
3. **導線設計**: Kaminokoe内で「面接対策完了 → 就活サービス紹介」の流れを検討

### 中期（送客サービス移行）
1. **提携候補への問い合わせ**
   - ネオキャリア（就職エージェントneo）
   - レバレジーズ（キャリアチケット）
   - i-plug（OfferBox）
2. **プライバシーポリシー整備**: 提携サービスへの情報提供に関する同意取得の仕組み
3. **一括登録UI設計**: デフォルトでチェックが入った同意フォーム
