# 転職・就活系アフィリエイトASP 詳細調査レポート

## 1. A8.net（エーハチネット）

### 登録手順
1. [A8.net公式サイト](https://www.a8.net/)の無料会員登録ページにアクセス
2. メールアドレスを入力し、「仮登録メールを送信」をクリック
3. 届いたメールの「登録用URL」をクリック
4. 以下の情報を入力：
   - ログインID（6〜20文字の半角英数字）
   - パスワード
   - 氏名・フリガナ
   - 電話番号
   - 生年月日・性別
   - 郵便番号・住所
5. 振込先口座情報を入力（ゆうちょ銀行が手数料最安でおすすめ）
6. サイトがない場合は無料ブログ「ファンブログ」のアカウントが自動作成

### 審査基準・審査期間
- **ASP登録審査：なし** — ブログやSNSがなくても登録可能
- **個別プログラム審査：あり** — 広告主ごとに独自の審査基準あり
- 「即時提携」マークの案件は審査なしで即座に広告掲載可能

### 支払い条件
| 項目 | 内容 |
|------|------|
| 最低支払額 | 1,000円〜5,000円（支払方式による） |
| 支払いサイクル | 月末締め → 翌々月15日払い |
| 振込手数料 | ゆうちょ銀行：66円、三井住友銀行：110〜220円、その他：495〜660円 |

### 支払方式
- 5,000円支払方式（デフォルト）
- 1,000円支払方式
- キャリーオーバー方式（繰越し）

### 転職系案件
- 300以上の人材系プログラムが稼働
- 転職エージェント、転職サイト、専門職向けサービスなど多数

**参考**: [A8.net公式サポート](https://support.a8.net/as/payment/)、[A8キャンパス](https://www.a8.net/campus/)

## 2. afb（アフィb）

### 登録手順
1. [afb公式サイト](https://www.afi-b.com/)にアクセス
2. 「無料でパートナー登録する」をクリック
3. 利用規約に同意し「同意して会員登録」
4. メールアドレスを入力し仮登録
5. 以下の情報を入力：
   - ユーザー情報
   - ログインID・パスワード
   - サイト情報（URL必須）
   - 口座情報
6. 審査完了後、SMS・メール認証で本人確認

### 審査基準・審査期間
- **サイト審査：あり**
- 審査基準：
  - コンテンツが充実していること
  - 公序良俗に反しないこと
  - コピーコンテンツでないこと
  - 会員制サイト（パスワード必要）は不可
- 審査期間：数日〜1週間程度
- 再審査：何度でも可能（落ちた理由を問い合わせ可能）

### 支払い条件
| 項目 | 内容 |
|------|------|
| 最低支払額 | **777円**（業界最低水準） |
| 支払いサイクル | **翌月末払い**（業界最速） |
| 振込手数料 | **無料**（afb負担） |
| 消費税 | **報酬額＋10%上乗せ**で支払い |

### 転職系案件
- 美容系に強いが、幅広いジャンルに対応
- dodaなど大手転職サービスの案件あり
- ビズリーチの案件取り扱いあり

**参考**: [afb公式](https://www.afi-b.com/)、[afb登録ガイド](https://yuka001.com/affiliate/afb-line/)

## 3. バリューコマース

### 登録手順
1. [バリューコマース公式](https://www.valuecommerce.ne.jp/)にアクセス
2. 必要なもの：ブログ/サイト（またはYouTube/Instagram）、銀行口座、メールアドレス
3. 会員登録フォームで基本情報を入力
4. サイト情報を登録
5. 審査完了を待つ

### 審査基準・審査期間
- **サイト審査：あり**
- 審査基準：
  - コンテンツが充実していること（目安：10〜20記事程度）
  - 運営者利用規約の遵守
  - 未成年者の運営は不可
  - 公序良俗・法令違反サイトは不可
- **審査期間：翌日〜3営業日**
- 再審査：修正後に再申し込み可能

### 支払い条件
| 項目 | 内容 |
|------|------|
| 最低支払額 | **1,000円** |
| 支払いサイクル | 承認月の**翌々月15日**払い |
| 振込手数料 | **無料**（全銀行対応） |

### 転職系案件
- Yahoo!ショッピング独占で有名だが、転職・金融系にも強い
- ビズリーチ案件あり（報酬：5,900〜6,600円）
- 就職・転職・アルバイト特集ページあり

**参考**: [バリューコマース公式](https://www.valuecommerce.ne.jp/)、[バリューコマース審査基準](https://yujiblog.org/valuecommerce-examination)

## 4. アクセストレード

### 登録手順
1. [アクセストレード公式](https://www.accesstrade.ne.jp/)にアクセス
2. メールアドレスを登録
3. 以下の情報を入力：
   - 個人情報
   - サイト/ブログ/SNSのURL
   - サイトPR（任意だが記入推奨）
   - 口座情報
4. 審査結果を待つ

### 審査基準・審査期間
- **サイト審査：あり**
- 審査基準：
  - ブログ：10〜15記事、各2,000文字以上が目安
  - Instagram：50投稿程度
  - オリジナルコンテンツであること
  - 他者のコンテンツのコピー不可
- **審査期間：3営業日程度**
- 再審査：問い合わせフォームから「登録再審査」を選択

### 支払い条件
| 項目 | 内容 |
|------|------|
| 最低支払額 | **1,000円** |
| 支払いサイクル | **翌々月15日**払い |
| 振込手数料 | **無料**（全金融機関対応） |

### 転職系案件（強み）
- **求人・金融系の高単価案件が充実**
- 転職・派遣系の案件が他ASPより多い
- doda案件あり
- 「アフィリエイト大学」で学習コンテンツ提供

**参考**: [アクセストレード公式](https://www.accesstrade.ne.jp/)、[アクセストレード登録ガイド](https://www.onamae.com/column/affiliate/47/)

## 5. もしもアフィリエイト

### 登録手順
1. [もしもアフィリエイト公式](https://af.moshimo.com/)にアクセス
2. 「会員登録する（無料）」をクリック
3. メールアドレスを入力し確認メール送信
4. 届いたURLをクリックし以下を入力：
   - ユーザーID・パスワード
   - 氏名・生年月日
5. 利用規約に同意して登録完了
6. サイト登録・金融機関登録を追加で行う

### 審査基準・審査期間
- **サイト審査：あり（比較的通りやすい）**
- 審査基準：
  - オリジナル記事が5記事以上
  - ガイドライン遵守
  - コピーコンテンツは不可
- 登録可能SNS：YouTube、Instagram、Twitter

### 支払い条件
| 項目 | 内容 |
|------|------|
| 最低支払額 | **1,000円**（住信SBIネット銀行なら**1円**から） |
| 支払いサイクル | 月末締め → **翌々月末**払い |
| 振込手数料 | **無料** |
| W報酬制度 | 通常報酬＋**12%ボーナス報酬** |

### 転職系案件
- 広告案件数は約1,800件（2024年12月時点）
- Amazonアソシエイトが利用可能（本家より審査通りやすい）
- 楽天などの有名広告も多数

**参考**: [もしもアフィリエイト公式](https://af.moshimo.com/)、[もしもアフィリエイト登録方法](https://www.xserver.ne.jp/blog/about-moshimo-affiliate/)

## 転職アフィリエイトを始める際の推奨順序

1. **A8.net** — 審査なしで即日登録可能。案件数最多で転職系も充実
2. **アクセストレード** — 求人・転職系に特化。doda案件あり
3. **バリューコマース** — ビズリーチ案件取得。審査も比較的通りやすい
4. **afb** — 消費税上乗せで実質報酬UP。支払いサイクル最速
5. **もしもアフィリエイト** — 初心者向け。W報酬制度でボーナスあり

## Kaminokoeへの活用示唆

### 推奨ASP登録順序
1. **A8.net**（審査なし）→ 即日登録・案件確認
2. **アクセストレード**（転職特化）→ 高単価案件多数
3. **バリューコマース**（ビズリーチ）→ ハイクラス層向け

### アフィリエイトリンク設置場所
- サービス利用後の「次のステップ」としての転職サイト紹介
- 面接対策完了後のCTA
- メール/プッシュ通知での転職サービス紹介

### 成果報酬条件の注意点
- 多くが「無料登録」で成果発生（成約ハードル低い）
- ビズリーチは年収500万円以上のみ成果対象
- 面談完了型は報酬高いが成果発生まで時間がかかる
