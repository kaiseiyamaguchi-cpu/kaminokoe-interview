## 評価軸
- **獲得単価**: 1件あたりの報酬額
- **工数**: 成果発生までのユーザー行動ハードル（低いほど良い）

## 詳細評価

### Sランク（高単価 × 低工数）
該当なし

---

### Aランク（高単価 × 中工数 or 中単価 × 低工数）

#### ビズリーチ
| 項目 | 詳細 |
|------|------|
| 単価 | **5,900〜6,600円**（高） |
| 成果条件 | 無料会員登録 + レジュメ登録完了 |
| 工数 | **中** — 登録のみで面談不要 |
| 制約 | 年収500万円以上のユーザーのみ成果対象 |
| 推奨ASP | バリューコマース |

**評価理由**: 高単価かつ面談不要。ただし年収制限があるためKaminokoeのターゲット層（就活生・第二新卒）との相性は要検討。ハイクラス転職者向けコンテンツがあれば有効。

---

### Bランク（高単価 × 高工数 or 低単価 × 低工数）

#### マイナビIT AGENT
| 項目 | 詳細 |
|------|------|
| 単価 | **10,000円**（最高） |
| 成果条件 | 面談完了 |
| 工数 | **高** — 登録→日程調整→面談実施が必要 |
| 推奨ASP | A8.net |

**評価理由**: 単価は最高だが、面談完了まで到達するユーザーは少ない。IT転職に特化した深いコンテンツがないと成約率が低くなる。

---

#### doda
| 項目 | 詳細 |
|------|------|
| 単価 | **4,760〜5,500円**（高） |
| 成果条件 | 登録から30日以内のカウンセリング来訪 |
| 工数 | **高** — 登録後にカウンセリング来訪が必要 |
| 推奨ASP | afb（消費税10%上乗せで実質最高） |

**評価理由**: 知名度が高く成約しやすいが、カウンセリング来訪が必要なため離脱リスクあり。登録だけでは成果にならない。

---

#### リクナビNEXT
| 項目 | 詳細 |
|------|------|
| 単価 | **約1,700円**（低） |
| 成果条件 | 新規会員登録のみ |
| 工数 | **低** — 登録だけで成果 |
| 推奨ASP | A8.net |

**評価理由**: 工数は最も低いが単価も低い。数を取る戦略なら有効だが、労力対効果は低め。

## Kaminokoe向け推奨

### 第一優先: ビズリーチ
- **理由**: 高単価 + 面談不要
- **課題**: 年収500万円以上の制限
- **対策**: 転職経験者・ミドル層向けコンテンツを追加

### 第二優先: リクナビNEXT
- **理由**: 登録のみで成果確定、ハードル最低
- **対策**: 単価が低いのでボリュームで稼ぐ導線設計

### 保留: doda / マイナビIT AGENT
- **理由**: 面談・カウンセリング必須で成果までの離脱が多い
- **再検討条件**: 面接対策→エージェント予約の一気通貫フローが構築できれば

## 補足: 専門職転職の高単価案件

一般調査によると、以下の領域は更に高単価：
- IT/エンジニア転職: 5,000〜30,000円
- 医療系転職: 10,000〜30,000円
- 看護師転職: 10,000〜50,000円

Kaminokoeの面接対策AIと相性が良い専門職があれば追加調査を推奨。
